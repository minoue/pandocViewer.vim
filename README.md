# vim_pandocViewer
Simple document viewer

## Requirements


## Install
* vim-plug
```
Plug minoue/vim_pandocViewer
```

## How to use
Open markdown file or any other pandoc-supported files.
Then run,
```
:PandocPreview
```

Make focus on a addrss bar of new window and hit <Enter>
